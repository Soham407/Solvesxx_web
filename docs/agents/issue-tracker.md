# Issue Tracker

This repo uses **GitHub Issues** for tracked work.

- Repository: `https://github.com/Soham407/Solvesxx_web`
- CLI: `gh`
- Default workflow: create issues in GitHub, label them, and link PRD coverage or audit notes from the issue body.

Use this tracker for:

- feature work
- cleanup tasks
- bug reports
- architecture refactors
- verification follow-ups

Avoid creating duplicate local issue files unless GitHub is unavailable.
