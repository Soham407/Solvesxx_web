# New Hook Guide

Before creating a new hook:
1. Verify it doesn't already exist by checking the 82 existing hooks in `.ai_context/CONTEXT.md`.
2. Review the DB schema in `.ai_context/SCOPE.md` section 14 to understand the data shape.
