# New Migration Guide

1. Read `.ai_context/SCOPE.md` section 14. Your new DB tables must follow the exact schema described there.
2. Read `.ai_context/CONTEXT.md` for standard conventions (e.g., RBAC policies).
3. Always ensure new tables integrate well with existing tables.
