# New Feature Guide

Before building a new feature or page, make sure to read the following files:
- `.ai_context/CONTEXT.md`
- `.ai_context/SCOPE.md`
- `.ai_context/PHASES.md`

- Check PHASES.md to see if it's already built or mocked.
- Follow the exact screens and API described in SCOPE.md.
